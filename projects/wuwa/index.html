WUWA Builds

September 2025 - Access at wuwa.thomasnewman.ca

In respect of this project, I wanted to make a personal website that I could use to publish my builds for the game "Wuthering Waves" (WUWA). I wanted to use the skills I've practiced making my other few websites to try to recreate some of the UI in the game. My website features the music in the game, the same clicking sounds, a custom loading screen, and animated backgrounds.

Features

  • Simple dark theme layout with CSS animations
  • Simple and clean loading animation using CSS and JS
  • Easily updatable for future builds to be added, and images and videos available for all characters in the game
  • Iconic music from the game, including click sounds and the cursor
