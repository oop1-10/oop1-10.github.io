Automatic Wall-Avoiding Robot

May 2025 | Completed in collaboration with Illia Siusiel

This project, called the Wall-Avoiding Robot (WAR) was created exclusively for the Jr. DEEP Saturdays Program, specifically the Tech Tinkerers program. The goal was to design a robot that could autonomously navigate around walls and obstacles, using sensors to detect its surroundings. It's final goal for the activity was to navigate it's way out of a maze using a preprogrammed algorithm. The robot is built using a Arduino Uno, an ultrasonic distance sensor, and a motor driver to control the movement of the wheels. The software is written in C++, utilizing the Arduino IDE.
